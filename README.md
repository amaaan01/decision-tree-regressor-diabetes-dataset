# Decision Tree Regressor on Diabetes Dataset

## Objective
Predict disease progression using patient health metrics.

## Dataset
Scikit-learn Diabetes Dataset.

## Model
Decision Tree Regressor

## Libraries
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Results

| Metric | Score |
|----------|----------|
| Test MSE | 3177.66 |
| Test R² Score | 0.4114 |
